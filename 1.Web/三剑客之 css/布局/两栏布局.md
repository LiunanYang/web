# 两栏布局
一般是左侧定宽栏和右侧自适应栏
## 方法
html结构
```html
<div class="parent">
  <div class="right">right</div>
  <div class="left">left</div>
</div>
```
### padding-left margin-left
```css
.parent{
  padding-left:-100px;
}
.left,.right{
 height: 500px;
 border: 1px solid;
 box-sizing: border-box;
 float: left;
}
.left{
width: 100px;
margin-left: -100%;
}
.right{
width: calc(100% - 100px);
margin-left: 100px;
}
```

###  flex
```css
.parent{
  display: flex;
}
.left,.right{
   height: 500px;
   border: 1px solid;
   box-sizing: border-box;
}
.left{
   width: 100px;
   flex: 0 0 100px;
   order:-1;
}
.right{
  width: 100%;
  flex-grow: 1;
}
```
