# css3 新增属性
## 布局
- flex
- grid
## 边框，盒子
- box-sizing：盒模型
- border-radius：边框圆角
- border-image：边框图像
## 背景
- background-size 背景图片的尺寸
- background-origin 背景图片从哪里开始绘制
- background-clip 背景图片的绘制区域
## 文本
- text-shadow: 文字阴影
- text-overflow：文字溢出处理
## 渐变
- linear-gradient 线性渐变
- radial-gradient 径向渐变
## 过渡 
- transition：过渡动画
## 动画
- animation: 自定义动画
## 转换 2d、3d
- transform:转换
